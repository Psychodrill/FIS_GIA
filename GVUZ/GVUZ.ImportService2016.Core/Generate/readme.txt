Чтобы обновить схемы:
1. скопировать в эту папку новые схемы с именами
import.xsd
import1app.xsd
delete.xsd

2. запустить 1.bat
если он автоматом не выполнит команду генерации классов по xsd, а просто откроет командную строку:
явно запустить generate.cmd

3. дальше эта команда сама создаст новые cs классы по схемам. Убедится, что они скопировались в папку dto\

4. открыть солюшн сервиса, убедится, что код компилится и работает

Enjoy!
