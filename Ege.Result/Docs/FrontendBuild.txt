Инструкция по сборке фронтэнда.

1. На машину необходимо установить NodeJs. Дистрибутив лежит на сервере \\SERVER\Install\Programming\JavaScript

2. Вместе с NodeJs ставится и npm, но его версия скорее всего устаревшая, поэтому нужно обновить npm.  
   В корне проекта App.Web (там, где лежит package.json и gulpfile.js) открываем консоль и пишем npm install npm -g

3. Устанавливаем все необходимые npm-пакеты:
   Там же, в корне App.Web в консоли: npm install
   Имена пакетов указывать не нужно, т.к. по умолчанию скачаются все те, которые указаны в package.json, но ещё не скачаны.
   В дальнейшем этой же командой можно докачать любые пакеты, которые в ходе разработки будут добавлены.

4. Устанавливаем gulp (глобально).
   В консоли: npm install gulp -g

5. Для запуска сборки в корне проекта (где есть gulpfile.js) в консоли пишем команду gulp


-------------------------

Возможно, возникнут ошибки, связанные с отсутствием прав на какие-то операции, в таком случае запустить консоль от админа.



--------------------------

Для добавления нового пакета в проект:


В той же папке, откуда запускаем gulp пишем команду в консоли:
npm install [имя пакета] --save-dev (или --save)

Флаги --save-dev и --save добавлять необходимо для того, чтобы зависимость автоматически прописалась в package.json и таким образом была сохранена в проекте.


-------------------------

Задачи getRegions и getAppealStatuses не запускаются при билде, при необходимости запускать вручную отдельно, например: gulp getRegions
