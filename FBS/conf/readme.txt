== Скрипты сборки приложения ==

Оформление .txt файлов в соответствии с языком, принятым в Media Wiki.

; Файлы NAnt с задачами для сборки проекта
* yachts.include.user.sample - файл настроек пользователя. Для работы необходимо переименовать в 
  файл yachts.include.user - выведенный из под контроля версий, и в нем установить свои локальные настройки
* yachts.include - настройки по умолчанию, используются для задания общих настроек и локальных, если не задан
  файл yachts.include.user
* yachts.build - основной файл задач
* db.include - задачи для сборки БД
